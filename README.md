# 🌈 VibeCheck - Yapay Zeka Destekli Sosyal Enerji Planlayıcısı

Future Talent 2026 - Yapay Zeka ile Ürün Geliştirme Programı Bitirme Projesi.

VibeCheck, kullanıcının anlık duygu durumuna (15 farklı mod) ve zaman kısıtına (30, 60, 120+ dk) göre en uygun mikro-aktivite planlarını optimize eden ve sunan akıllı bir web uygulamasıdır.

## 🚀 Proje Yapısı (Architecture)
* `/frontend` — Cıvıl cıvıl, hareketli baloncuklu, modern Design System mimarisine sahip kullanıcı arayüzü (HTML/CSS/JS).
* `/backend` — FastAPI ve Uvicorn tabanlı, LLM (Gemini API) entegrasyonuna hazır asenkron API motoru.
* `/prodocs` — Proje anayasası (PRD), Gelişim günlüğü, Teknik dokümantasyon ve Tasarım kuralları.

## 🛠 Kullanılan Teknolojiler (Tech Stack)
* **Frontend:** HTML5, CSS3 (Glassmorphism & Keyframe Animations), JavaScript (ES6+).
* **Backend:** Python 3.10+, FastAPI, Uvicorn, Python-dotenv.
* **AI Engine:** Gemini 1.5 Flash API (Hafta 5 Entegrasyonu).
